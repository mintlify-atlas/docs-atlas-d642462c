## Constructor

```typescript
import { RestClient } from 'okx-api';

const client = new RestClient({
  apiKey: 'YOUR_API_KEY',
  apiSecret: 'YOUR_API_SECRET',
  apiPass: 'YOUR_API_PASSPHRASE',
});
```

### Constructor options

`RestClient` accepts a single optional `RestClientOptions` object.

<ParamField body="apiKey" type="string">
  Your OKX API key. Required for authenticated endpoints.
</ParamField>

<ParamField body="apiSecret" type="string">
  Your OKX API secret. Required for authenticated endpoints.
</ParamField>

<ParamField body="apiPass" type="string">
  Your OKX API passphrase. Required for authenticated endpoints.
</ParamField>

<ParamField body="market" type="string" default="prod">
  The API group to connect to. Accepted values:
  - `'prod'` — OKX global (`www.okx.com`)
  - `'EEA'` — European Economic Area (`my.okx.com`)
  - `'US'` — United States (`app.okx.com`)
</ParamField>

<ParamField body="demoTrading" type="boolean" default="false">
  Set to `true` to use OKX demo trading. No real funds are used.
</ParamField>

<ParamField body="strict_param_validation" type="boolean" default="false">
  When `true`, throws an error if any request parameter is `undefined`.
</ParamField>

<ParamField body="baseUrl" type="string">
  Override the default API base URL (protocol + domain). Use `market` instead to target `my.okx.com`, `app.okx.com`, etc.
</ParamField>

<ParamField body="parse_exceptions" type="boolean" default="true">
  When `true`, the client attempts to post-process request exceptions for a cleaner error object.
</ParamField>

<ParamField body="keepAlive" type="boolean">
  Enable HTTP keep-alive for REST requests (via axios).
</ParamField>

<ParamField body="keepAliveMsecs" type="number" default="1000">
  Interval (ms) for TCP keep-alive packets. Only relevant when `keepAlive` is `true`.
</ParamField>

<ParamField body="customSignMessageFn" type="(message: string, secret: string) => Promise<string>">
  Provide a custom HMAC signing function. Useful for using Node's native `crypto.createHmac` for better performance. See the `examples/` folder for a demonstration.
</ParamField>

| Method | Auth | Description |
|--------|:----:|-------------|
| `fetchLatencySummary()` | No | Measures round-trip latency and clock drift between client and OKX servers. Logs a warning if drift exceeds 500 ms. |
| `getServerTime()` | No | Returns the current OKX server time as a Unix timestamp (ms). |
