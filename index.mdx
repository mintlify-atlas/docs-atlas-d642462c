The `okx-api` package is a fully-featured, actively maintained Node.js and TypeScript SDK for interacting with the [OKX exchange](https://www.okx.com). It covers every endpoint in OKX's REST API and provides a robust, auto-reconnecting WebSocket client for real-time market data and private account streams.

## Installation

<CodeGroup>
```bash npm
npm install okx-api
```
```bash yarn
yarn add okx-api
```
```bash pnpm
pnpm add okx-api
```
</CodeGroup>

## Key exports

```typescript
import {
  RestClient,           // Full REST API client
  WebsocketClient,      // Real-time WebSocket streams
  WebsocketAPIClient,   // Promise-based WebSocket API (orders)
} from 'okx-api';
```
